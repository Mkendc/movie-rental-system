# Movie Rental System

A Java-based Movie Rental System developed during TESDA training.

## Features
- Customer Management
- Movie Inventory
- Staff Management
- Login System
- Database Integration

## Technologies Used
- Java
- MySQL
- Swing GUI
- JDBC

## Database Setup
1. Open MySQL
2. Import SQL files inside the SQL folder
3. Run the SQL scripts

## How to Run
1. Open project in NetBeans
2. Configure database connection
3. Run LoginForm1.java
